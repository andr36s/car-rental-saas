# car-rental-saas
Este proyecto propone el desarrollo de una plataforma web para la gestión de procesos de alquiler de vehículos. La solución busca centralizar y digitalizar las operaciones principales de una empresa de renta de vehículos, permitiendo la gestión eficiente de clientes, empleados, vehículos, reservas y contratos.

## Objetivo 
Evaluar la capacidad del equipo para abstraer un problema empresarial y estructurar una propuesta arquitectónica inicial, sustentada técnicamente y respaldada por fundamentos teóricos.

## Descripción de la actividad
Los estudiantes, organizados en equipos, desarrollan el diseño arquitectónico inicial del sistema definido como proyecto de aula.

## Descripción del problema
Las empresas para el alquiler de vehículos, gestionan sus procesos de manera fragmentada o con sistemas partidos, lo que genera perdida de trazabilidad o reprocesos innecesarios para los clientes además de ir presencialmente. Esta empresa de alquiler de vehículos contará con empleados operativos y clientes externos que realizan su registro de manera remota y así los clientes solo se acercan a recoger el vehículo.

Actualmente, la empresa realiza el proceso de creación de reservas de manera presencial mediante el diligenciamiento de formularios físicos. De igual manera, los contratos de arrendamiento se gestionan exclusivamente en formato físico, lo que obliga a los clientes a presentarse personalmente en las instalaciones de la empresa para firmar los documentos correspondientes.

Además, el proceso de alquiler requiere que los usuarios interactúen con al menos tres empleados diferentes para completar toda la gestión del servicio, lo que genera demoras, reprocesos y una mayor probabilidad de errores en el manejo de la información.

La necesidad que se va a cubrir es centralizar y convertir a remoto la gestión de procesos de reservas y contratos en plataforma web con acceso por roles (Empleado, Cliente). Esperamos con esto una reducción en la pérdida de información, errores manuales y mayor satisfacción al realizar los procesos a distancia.
